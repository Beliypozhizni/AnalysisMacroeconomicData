# Macroeconomic Data Analyzer

CLI-утилита для анализа макроэкономических данных из CSV-файлов и генерации отчётов.

---

## Установка зависимостей

В проекте для управления зависимостями используется Poetry.

Следует создать виртуальную среду и активировать её (Windows):

```
python -m venv .venv
.\.venv\Scripts\activate
```

Установить зависимости:

```
poetry install
```

## Использование

Пример:

```
python -m macro_analyzer --files economic1.csv economic1.csv --report average-gdp
```

## Тесты

Зависимости для тестов находятся в группе "dev"

Запуск тестов с покрытием:

```
pytest
```

## Добавление новых отчетов

В <i><b>macro_analyzer/reports</b></i> следует создать новый .py файл, 
в котором нужно реализовать класс с полем <i><b>name</b></i> и
методом <i><b>build</b></i>, чтобы соответствовать протоколу, 
объявленному в <i><b>macro_analyzer/reports/base</b></i>.

Затем нужно добавить экземпляр данного класса в словарь <i><b>\_REPORTS</b></i>,
находящемся в <i><b>macro_analyzer/reports/\_\_init__</b> по аналогии с существующим отчетом.
